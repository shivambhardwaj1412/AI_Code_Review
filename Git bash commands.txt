# If git hub token expired:- 

The GITHUB_TOKEN in your .env is expired or invalid. The bot received the webhook but couldn't fetch the PR diff from GitHub because the token was rejected.

Fix — Generate a New GitHub Token
Go to → github.com/settings/tokens

Click "Generate new token (classic)"

Give it these permissions:

✅ repo (full)

✅ write:discussion

Click Generate token → copy it

Update your .env:

Restart the server:
